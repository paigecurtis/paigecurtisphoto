# Where to Put Your Images

**Put every image file inside the `website/images/` folder.**  
The site looks for exact filenames. If the name matches, the image appears in the right spot automatically.

---

## Step 1: Open the images folder

On your computer, go to:

```
CursorOfflineMap → website → images
```

That’s the only folder that holds your photos for the site.

---

## Tips

- **Format:** JPG is best. PNG is fine too; if you use PNG, rename the file in the list above from `.jpg` to `.png` and update the same filename in the HTML (or keep the filename as `.jpg` and export/save as JPG).
- **Naming:** Names are case-sensitive. Use lowercase and the exact spelling: `meet-paige.jpg`, not `Meet-Paige.jpg`.
- **Missing images:** If a file isn’t in `images/`, a grey placeholder shows. Add the file with the right name and refresh the page.
- **Same image in two places:** You can copy the same file with two different names (e.g. copy `my-photo.jpg` to `gallery-1.jpg` and `port-couples-1.jpg`) so one photo appears in multiple spots.

Once the files are in `website/images/` with these names, they’ll show up in the right spots on the site.
